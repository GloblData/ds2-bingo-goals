--How the "Random Starting Class" works--

Roll a D8, depending on the number you must pick one of the following:
   
  1. Warrior 
  2. Knight
  3. Swordsman
  4. Bandit
  5. Cleric
  6. Sorcerer
  7. Explorer
  8. Deprived
   
--How killing a boss with a random spell type works--

Roll a D4 to determine which spell type you need to use:

  1. Miracles
  2. Sorcery
  3. Pyromancy
  4. Hexes

If you happen to have a square restricting that spell type, you may ignore it 
for this fight only

--Picking your own Square--

For this, you can pick any challenge from the list (something advantageous to
your route, for example).
